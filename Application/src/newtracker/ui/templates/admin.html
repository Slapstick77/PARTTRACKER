Automatic import

Automatic imports use the same unified import process as the manual run.

That means new job files in Programming folders are included, not just .DAT files.

Amada, EMK1, and Laser scan only the selected folder. Programming folders still scan subfolders.

Files older than 6 months are skipped, and files changed within the last 2 minutes are deferred until they are stable enough to import safely.

Use 0 to disable scheduled automatic imports.

Main scanner automation

Choose what happens right after a DAT loads expected parts on the Main Scanner page.

Off keeps the current flow. Auto Complete moves parts only. Full Auto also finishes the batch.
